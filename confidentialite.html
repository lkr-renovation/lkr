Politique de confidentialité

Ce site collecte uniquement les données que vous nous envoyez via le formulaire de contact.

  • Finalité : répondre à votre demande d’information.
  • Base légale : intérêt légitime de LKR RENOVATION MC SARL.
  • Conservation : les données ne sont pas stockées dans une base du site. Elles sont transmises par email à info@lkr.mc.
  • Destinataires : l’équipe interne de LKR RENOVATION MC SARL.
  • Droits : accès, rectification, suppression. Contact : info@lkr.mc.

Lorsque l’option d’envoi par email (mailto) est utilisée, les informations transitent par votre logiciel de messagerie. Lorsque vous utilisez le formulaire, les données sont traitées via un prestataire d’emailing (Resend) uniquement pour l’acheminement de votre message.
